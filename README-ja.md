# D-pad File Explorer

[English](./README.md) | **日本語**

Linuxターミナル向けの、高速なキーボード駆動（矢印キー）ファイルエクスプローラーです。

ディレクトリを開くために `Enter` を押し、戻るためにコマンドを入力する必要がある従来のファイルマネージャーとは異なり、D-pad File Explorerは4列の「ミラーカラム（Miller columns）」レイアウトを採用しています。矢印キーのみを使用してファイルシステム全体を簡単にナビゲートできるため、ディレクトリの移動が劇的に速くなります。

## 特徴

- **直感的な矢印キーナビゲーション**: `上`/`下`で選択、`右`で中に入り、`左`で戻ります。
- **ミラーカラムレイアウト**: 祖父母、親、現在、子のディレクトリを横並びに表示します。現在の場所と移動可能な場所が常に把握できます。
- **動的なリサイズ**: ターミナルウィンドウのサイズに基づいて、表示されるファイル数と列の幅を自動的に調整します。
- **シェル統合 (`cd`)**: ディレクトリで `Enter` を押すと、即座にエクスプローラーを終了し、ターミナルでそのパスに `cd` します。
- **素早いファイル展開**: ファイルで `Enter` を押すと、ターミナル内で直接 `vi` を使用して内容を表示および編集できます。
- **即座のファイルプレビュー**: フォーカスされている項目がファイルの場合、4列目にその内容のライブプレビューがインテリジェントに表示されます（バイナリファイルや読み取れないファイルは安全にスキップされます）。
- **隠しファイル**: 隠しファイルおよびディレクトリの表示を完全にサポートします。

## 要件

- **OS**: Linux / macOS (標準的なPOSIXターミナル環境)
- **コンパイラ**: `g++` (C++17以上)
- **ライブラリ**: `ncurses` (ワイド文字サポートが必要)

### 依存関係のインストール (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install -y g++ make libncurses5-dev libncursesw5-dev
```

## インストールとビルド

リポジトリをクローンして `make` を実行します：

```bash
git clone https://github.com/goroshirow/dpad-file-explorer.git
cd dpad-file-explorer
make
```

これにより `dpad_explorer` 実行可能ファイルが生成されます。

## 初期設定 (`cd`機能に必須)

ディレクトリで `Enter` を押して実際のシェルのカレントディレクトリを変更するという、このツールのコア機能を有効にするには、**必ず**シェルラッパー関数を使用する必要があります。子プロセス（このC++プログラム）が親シェルの作業ディレクトリを直接変更することはできないためです。

また、この関数を定義することで、わざわざ実行ファイルを `bin` に配置しなくても、どこからでも `dpad` コマンドでエクスプローラーを呼び出せるようになります。

`~/.bashrc`（または `~/.zshrc`）に以下を追加してください：

```bash
function dpad() {
    # コンパイルした実行ファイルの絶対パスを指定してください
    # 例: local dest=$(/home/user/dpad-file-explorer/dpad_explorer)
    local dest=$(/path/to/dpad_explorer)
    
    if [ -n "$dest" ]; then
        cd "$dest"
    fi
}
```

追加後、シェルの設定を再読み込みします：

```bash
source ~/.bashrc
```

これで、ターミナルで `dpad` と入力するだけでどこからでもエクスプローラーが起動し、終了時に選択したディレクトリへ移動できるようになります。

## キーバインドとコマンド

| キー | アクション |
| --- | --- |
| `↑` (上) | 現在のディレクトリで選択を上に移動 |
| `↓` (下) | 現在のディレクトリで選択を下に移動 |
| `→` (右) | **ディレクトリ上:** 選択したディレクトリに入る。<br>**ファイル上:** プレビュー列にフォーカスし、内容をスクロールできるようにする。 |
| `←` (左) | 親ディレクトリに戻る（またはプレビューのフォーカスを解除する）。 |
| `Enter` | **ディレクトリ上:** エクスプローラーを終了し、そのディレクトリに `cd` する。<br>**ファイル上:** `vi` でファイルの内容を表示/編集する。 |
| `/` (スラッシュ) | **検索モード:** 入力を開始して、現在のディレクトリ内をインクリメンタル検索する。 |
| `:` (コロン) | **コマンドモード:** コマンドを入力して実行する（下記参照）。 |
| `ESC` | 検索モード、コマンドモード、またはプレビューのフォーカスを終了する。 |
| `q` または `Q` | ディレクトリを変更せずにエクスプローラーを終了する。 |

### コマンドモード (`:`)

`:` を押してから以下のコマンドを入力し、`Enter` を押します。

| コマンド | 説明 |
| --- | --- |
| `R` または `r` | フォーカスしている項目の**名前を変更** (Rename) します。 |
| `D` または `d` | フォーカスしている項目を**削除** (Delete) します。空でないディレクトリの場合は確認プロンプトが表示されます。 |
| `md` | 現在のディレクトリに新しい**ディレクトリを作成** (Make Directory) します。 |
| `mf` | 現在のディレクトリに新しい**ファイルを作成** (Make File) します。 |
| `unzip` | フォーカスしているzipファイルを現在のディレクトリに**解凍** (Unzip) します。 |
